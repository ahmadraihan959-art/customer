# SCM Customer - NestJS

## Deskripsi

Project ini merupakan scaffolding (kerangka) backend menggunakan NestJS yang dibuat berdasarkan konsep Supply Chain Management (SCM) dari sisi customer.

## Alur Sistem

Customer login → melihat produk → memilih produk → melakukan pemesanan → melihat status pesanan

## Struktur Module

- auth: menangani login customer
- users: menyimpan data customer
- products: menampilkan produk/jasa
- orders: mengelola pemesanan dan status pesanan

## Penjelasan

Sistem ini dirancang berdasarkan alur SCM dimana customer sebagai pengguna utama dapat melakukan proses pemilihan layanan hingga pemesanan dan monitoring status pesanan.

Project ini hanya berupa struktur dasar (scaffolding) tanpa implementasi logika bisnis.
